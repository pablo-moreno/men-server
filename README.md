# Mongo, Express, Node Server

[![Build Status](https://travis-ci.com/pablo-moreno/men-server.svg?branch=master)](https://travis-ci.com/pablo-moreno/men-server)

Install dependencies:

```bash
npm i
```

Development server:

```bash
npm run dev
```

Build and serve:

```bash
npm run serve
```

Run tests:

```bash
npm run test
```
